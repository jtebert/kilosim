Kilosim::LightPattern Class Reference

Detailed Description

A LightPattern represents the visible light intensity of the World. This makes the simplifying assumption that perceived light intensity from the sensors is linearly proportional to the luminosity. Any non-monochrome images will be converted to grayscale by a luminosity for computing light intensity.
